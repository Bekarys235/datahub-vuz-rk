# DataHub — Top 25 Universities RK

Интерактивный сайт с рейтингом 25 вузов Казахстана, сравнением университетов, фильтрами, поиском и 3D-туром для Nazarbayev University.

## Структура проекта
- `index.html` — интерфейс сайта (HTML + CSS + JS).
- `data/universities.json` — данные о 25 университетах.
- `README.md` — описание проекта.
- `.gitignore` — список файлов и папок, исключаемых из Git.

## Как запустить локально
### Вариант 1 — через Python:
python -m http.server 8000

Потом открой в браузере:
http://localhost:8000

### Вариант 2 — через Node.js:
npx serve .

## Обновление данных
Редактируйте файл:
data/universities.json

После изменения:
git add .
git commit -m "Update data"
git push origin main

## 3D-тур
Для Nazarbayev University установлена кнопка, которая открывает официальный 3D-тур:
https://nu.edu.kz/ru/campus/campustour
